# Analysis-of-the-Relationship-Between-Urban-Spaces-and-Biosignals-in-Young-Adults

Bachelor's Project in Biomedical Engineering, University of Coimbra.

## Description
This project investigates the relationship between urban spaces and physiological, behavioral and psychological variables in young adults using wearable sensing data.

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- OpenPyXL

## Repository Structure
1. Sample Characteristics
2. Physical Activity Patterns
3. Urban Spaces
4. Relationship Between Variables
5. Geospatial RMSSD Map
